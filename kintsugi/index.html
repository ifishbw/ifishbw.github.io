Initial Game Designs

Camera System

My design goals I was able to acheive for the camera system I developed.

  • Robust enough that the player would never have to control the camera manually
  • Smooth enough that people who get motion sick would have no issues
  • Allow for easy implementation of level tools to enable further control camera
  • Design for Controller experience

A lot of the design was based on a GDC talk by the gamefeel engineer for Journey.

Movment System

During Development I took over the movement system and implemented:

  • Refactored all the code into a State Machine.
  • Implemented Mechanics swoop and mantle.
